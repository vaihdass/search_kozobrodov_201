## Задание 3. Инвертированный индекс и булев поиск

Инвертированный индекс по леммам + булев поиск с AND, OR, NOT и скобками.

### Структура проекта

```
hw3/
    index_builder.py      # построение инвертированного индекса
    bool_parser.py        # парсер булевых запросов
    search.py             # точка входа, принимает запрос из аргументов
    inverted_index.json   # результат: {лемма: [doc_ids]}
```

### Пример

```bash
python3 search.py "курица AND рис"
# Найден 1 документ: 0014

python3 search.py "(курица AND рис) OR (рыба AND картофель)"
# Найдено 4 документа: 0014, 0021, 0061, 0111
```

При некорректном запросе парсер выводит ошибку:

```bash
python3 search.py "(курица AND"
# Ошибка: неожиданный конец запроса

python3 search.py "(курица AND рис"
# Ошибка: ожидался ')', получен 'None'
```

### Зависимости

```bash
pip install beautifulsoup4 lxml pymorphy3 nltk
```

### Запуск

```bash
cd hw3
python3 index_builder.py          # построить индекс
python3 search.py "<запрос>"      # выполнить поиск
```


## Задание 4. TF-IDF

Для каждого документа считаем TF-IDF по токенам и по леммам.

### Формулы

Для токенов (конкретных форм слова):
- TF = кол-во вхождений токена в документ / общее кол-во токенов в документе
- IDF = log(кол-во документов / кол-во документов, содержащих этот токен)

Для лемм (нормальных форм):
- TF = суммарное кол-во вхождений всех словоформ леммы / общее кол-во токенов в документе
- IDF = log(кол-во документов / кол-во документов, содержащих хотя бы одну словоформу леммы)

TF-IDF = TF * IDF (чем выше, тем слово важнее для документа)

### Структура проекта

```
hw4/
    tfidf.py        # подсчет TF-IDF
    tokens/         # результат по токенам (179 файлов)
    lemmas/         # результат по леммам (179 файлов)
```

Формат выходных файлов: `<термин> <idf> <tf-idf>`

### Пример

tokens/0001.txt:
```
большого 4.494239 0.011985
вашей 2.789491 0.007439
взгляд 2.622436 0.006993
```

lemmas/0001.txt:
```
большой 1.549800 0.004133
ваш 1.275363 0.003401
взгляд 2.622436 0.006993
```

```bash
pip install beautifulsoup4 lxml pymorphy3 nltk
```

### Запуск

