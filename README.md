## Задание 3. Инвертированный индекс и булев поиск

Инвертированный индекс по леммам + булев поиск с AND, OR, NOT и скобками.

### Структура проекта

```
hw3/
    index_builder.py      # построение инвертированного индекса
    bool_parser.py        # парсер булевых запросов
    search.py             # точка входа, принимает запрос из аргументов
    inverted_index.json   # результат: {лемма: [doc_ids]}
```

### Пример

```bash
python3 search.py "курица AND рис"
# Найден 1 документ: 0014

python3 search.py "(курица AND рис) OR (рыба AND картофель)"
# Найдено 4 документа: 0014, 0021, 0061, 0111
```

При некорректном запросе парсер выводит ошибку:

```bash
python3 search.py "(курица AND"
# Ошибка: неожиданный конец запроса

python3 search.py "(курица AND рис"
# Ошибка: ожидался ')', получен 'None'
```

### Зависимости

### Запуск

```bash
cd hw3
python3 index_builder.py          # построить индекс
python3 search.py "<запрос>"      # выполнить поиск
```
